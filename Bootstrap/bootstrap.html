HTML
<HTML>

Html is an essential tag, which tells the web browser that we're about to insert some html here.

HEAD
<HEAD>

Head tells the browser some essential information like in what address is the css style and javascript files. There's also the character set and some meta-information.

BODY
<BODY>

Body contains the actual seen web page and it's properties.

P
<P>

P stands for "paragraph" and can be used to insert text into html page.

IMG
<IMG>

With IMG images, pictures or photographs can be shown on a web page.

A
<A>

A makes a link to somewhere, either to an external content or a local one.

BR
<BR>

BR creates a line break on a webpage.

H1
<H1>

H1 is used to make a big heading text.
